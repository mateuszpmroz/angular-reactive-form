Registration form

Name field is required.
Surname field is required.
Email field is required. Value must be type of email.
Value must be type of phone number (numbers, '+', '-', '(', ')').

Password field is required.
Password must contain at least one upper case character and one of the @#$%^& characters.
Password must be min 8 character length.
Password and repeat password must be the same.

Dog Cat Other Pet field is required.

City field is required.
Street field is required.
Building field is required.
